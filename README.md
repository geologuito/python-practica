# 🐍 Python Práctica

Repositorio personal de práctica de programación en Python.
Incluye ejercicios, guías y experimentos realizados durante el aprendizaje progresivo del lenguaje.

---

## 🎯 Objetivo

Desarrollar una base sólida en programación utilizando Python, enfocándome en:

* Pensamiento lógico y resolución de problemas
* Uso de estructuras de control (condicionales y bucles)
* Manejo de estructuras de datos
* Escritura de código limpio y organizado

## 📚 Contenidos del curso

El aprendizaje se desarrolla de forma progresiva:

### 🟢 Fundamentos

* Variables y tipos de datos
* Entrada y salida (`input` / `print`)
* Operadores

### 🟡 Control de flujo

* Condicionales (`if`, `elif`, `else`)
* Formateo de strings (f-strings)
* Bucles `while` y `for`
* Validación de datos

### 🟠 Estructuras de datos

* Listas y tuplas
* Diccionarios

### 🔵 Modularidad

* Funciones
* Módulos

### 🟣 Persistencia de datos

* Archivos
* Bases de datos (SQLite)
* Operaciones CRUD

### 🔴 Proyecto final

* Desarrollo de una aplicación completa integrando todos los conceptos
